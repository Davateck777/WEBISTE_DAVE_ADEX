# Assets Folder

Place your images and media files here.

## Recommended Files

### Profile Image
- **Filename:** `profile.jpg` or `profile.png`
- **Size:** 400x400px minimum (square)
- **Format:** JPG or PNG
- **File Size:** Under 500KB (compress if needed)

### Other Images
- Project screenshots
- Company logos
- Certificates
- Any other visual assets

## Image Optimization Tips

1. **Compress images** before uploading
   - Use [TinyPNG](https://tinypng.com)
   - Use [Squoosh](https://squoosh.app)

2. **Use appropriate formats:**
   - JPG for photos
   - PNG for logos/graphics with transparency
   - SVG for icons (scalable)

3. **Optimize dimensions:**
   - Don't upload images larger than needed
   - Resize to actual display size
